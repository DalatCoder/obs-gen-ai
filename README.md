## 🛠️ Công cụ và Framework

### 📝 Framework Prompting (TCREI)

> 🔗 **Chi tiết**: [[Google Prompting Essentials/01 - 05 - Use the 5-step prompt framework]] | [[Google Prompting Essentials/01 - 08 - Get the most out of AI with the prompt framework]]

```
Task (Nhiệm vụ) → Context (Ngữ cảnh) → References (Tham khảo) → Evaluate (Đánh giá) → Iterate (Lặp lại)
```

### 🔍 Phương pháp ABI (Always Be Iterating)

> 🔗 **Tìm hiểu thêm**: [[Google Prompting Essentials/01 - 09 - Practice 4 iteration methods]] | [[Google Prompting Essentials/01 - 10 - Generate better outputs through iteration]]

- Luôn luôn điều chỉnh và cải thiện prompt
- Không bỏ cuộc sau lần thử đầu tiên
- Học hỏi từ mỗi kết quả để cải thiện

- Xác định vai trò cụ thể cho AI → [[Google Prompting Essentials/Glossary - Thuật ngữ Gen AI#Persona|Tìm hiểu Persona]]
- Thiết lập đối tượng nhận thông tin
- Định dạng output phù hợp với mục đích

### 🔗 Kỹ thuật Prompt Chaining nâng cao

> 🔗 **Tìm hiểu**: [[Google Prompting Essentials/04 - 02 - Discover prompt chaining]] | [[Google Prompting Essentials/04 - 04 - Simplify multi-step projects with prompt chaining]]

- **Prompt Chaining** - kết nối các prompt liên tiếp để xử lý tác vụ phức tạp → [[Google Prompting Essentials/Glossary - Thuật ngữ Gen AI#Prompt Chaining|Tìm hiểu Prompt Chaining]]
- **Chain-of-Thought** - hướng dẫn AI giải thích từng bước suy luận → [[Google Prompting Essentials/Glossary - Thuật ngữ Gen AI#Chain of Thought|Chain of Thought]]
- **Tree-of-Thought** - khám phá nhiều giải pháp song song và đánh giá → [[Google Prompting Essentials/Glossary - Thuật ngữ Gen AI#Tree of Thought|Tree of Thought]]
- **Meta-prompting** - sử dụng AI để tối ưu hóa và tạo prompt mới → [[Google Prompting Essentials/Glossary - Thuật ngữ Gen AI#Meta-Prompting|Meta-Prompting]]

- **Multi-step Reasoning** - suy luận đa bước cho bài toán logic phức tạp
